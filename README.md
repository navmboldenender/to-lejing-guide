# to-lejing-guide

本仓库用于归档和发布若干独立 HTML 页面。这些页面均为静态内容，不依赖后端服务，可直接通过浏览器打开或部署到任何支持静态文件的托管平台。

## 项目简介

这是一个以存档和展示为目的的仓库，主要存放一些独立的 HTML 页面。每个页面都是自包含的，可以单独访问。仓库本身不对应任何特定网站或域名，仅作为页面集合的载体。

## 目录说明

```
to-lejing-guide/
├── index.html          # 仓库主页（可选）
├── pages/              # 独立 HTML 页面存放目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/             # 公共资源（样式、脚本、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/` 目录用于存放各个独立的 HTML 页面。
- `assets/` 目录包含页面可能共用的样式、脚本或图片资源。
- 每个页面应尽量保持独立，避免依赖外部资源（除非在仓库内引用）。

## 页面归档说明

- 所有页面均为静态 HTML，可在本地直接打开运行。
- 页面内容不涉及任何具体商业、营销或推广信息。
- 页面功能、样式、用途由各页面自身定义。
- 仓库维护者会不定期更新或添加新的页面。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来修正错误、改进页面或增加新内容。
- 请确保新增页面为纯静态 HTML，不包含恶意代码、跟踪脚本或外部依赖（除非必要）。
- 仓库管理员会审核所有合并请求，并保留对内容的最终解释权。
- 本仓库不保证页面的持续可用性或兼容性，但会尽力保持基本可用状态。
